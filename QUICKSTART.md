# Быстрый старт

## Установка

```bash
# 1. Установите зависимости для всех модулей
npm run install:all

# Или по отдельности:
cd bot && npm install
cd ../backend && npm install
cd ../mini-app && npm install
```

## Настройка

1. Скопируйте `.env.example` в `.env` в корне проекта
2. Заполните `BOT_TOKEN` (получите у @BotFather)
3. Настройте `NEXT_PUBLIC_API_URL` (URL вашего backend)

## Запуск в режиме разработки

```bash
# Запуск всех сервисов одновременно
npm run dev

# Или по отдельности:
npm run dev:bot      # Telegram Bot (порт по умолчанию)
npm run dev:backend  # Backend API (порт 3000)
npm run dev:app      # Mini App (порт 3001)
```

## Тестирование

1. Откройте Telegram и найдите вашего бота
2. Отправьте `/start`
3. Откройте Mini App через бота
4. Создайте тестовую заявку
5. Проголосуйте за канал

## Структура данных

Данные хранятся в `backend/data/`:
- `requests.json` - все заявки
- `users.json` - пользователи и их токены BOOST

## Важные замечания

- Токены BOOST хранятся офчейн (в JSON файлах)
- 1 голос = 1 BOOST токен
- Токены блокируются до окончания срока голосов
- После разблокировки пользователь может вывести токены
